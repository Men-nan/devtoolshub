## 文章内容

大家好，今天给大家分享一个我开源的在线开发者工具集合 - **DevToolsHub**。

### 🎯 为什么做这个项目？

作为开发者，我们经常需要：
- 格式化 JSON 数据
- 编解码 Base64
- 测试正则表达式
- 生成 UUID、密码
- 转换颜色格式
- ...

虽然有很多在线工具，但大多数都有这些问题：
- ❌ 广告太多，影响使用
- ❌ 界面老旧，体验差
- ❌ 需要注册，不安全
- ❌ 数据上传，有隐私风险

所以我做了 **DevToolsHub**，一个简洁、安全、免费的在线工具集合。

### ✨ 项目亮点

#### 1. 🔒 隐私安全
所有数据在浏览器本地处理，不上传任何数据到服务器。无需注册，无需登录。

#### 2. ⚡ 极速体验
纯静态页面，加载速度极快。响应式设计，完美适配移动端。

#### 3. 🎨 现代设计
简洁美观的 UI 界面，流畅的交互动画，专业的开发者体验。

### 🛠️ 包含工具

| 工具 | 说明 |
|------|------|
| 📄 JSON 格式化 | 格式化、校验、压缩 JSON 数据 |
| 🔐 Base64 编解码 | 文本 Base64 编码与解码 |
| 🔗 URL 编解码 | URL/URI 组件编码与解码 |
| 🔢 哈希生成器 | SHA-1/SHA-256/SHA-384/SHA-512 哈希 |
| 🧪 正则测试器 | 实时正则表达式匹配与高亮 |
| 🎨 颜色转换器 | HEX/RGB/HSL 颜色格式互转 |
| 🆔 UUID 生成器 | 批量生成 UUID v4 |
| 🔑 密码生成器 | 可配置强度的安全密码生成 |
| ⏰ 时间戳转换 | Unix 时间戳与日期互转 |
| 📝 文本对比 | 文本差异对比与高亮 |

### 🛠️ 技术栈

- **Next.js 16** (App Router)
- **TypeScript**
- **Tailwind CSS 4**
- **GitHub Pages** 部署
